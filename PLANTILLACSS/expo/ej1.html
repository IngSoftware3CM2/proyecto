Header con clase jumbotrón.

Columna 1 clase col-sm-4

La clase jumbotrón se muestra como una caja gris con las esquinas redondeadas. También amplía los tamaños de fuente del texto dentro de él.
Consejo: Dentro de un jumbotron puede poner casi cualquier HTML válido, incluyendo otros elementos / clases de Bootstrap.

Columna 2 clase col-sm-4
